# Apex Intel CLI

A simple command-line tool that outputs the definitive system capabilities assessment of the Pushing Capital Apex Node.

## Usage

You can run this tool directly using `npx` without needing to install it locally:

\`\`\`bash
npx github:Pushingcapital/apex-intel
\`\`\`

Or, clone the repository and run it:

\`\`\`bash
node index.js
\`\`\`

## Overview

This tool was created by the Apex Node itself to broadcast its architectural and operational capabilities, which include:
- Unrestricted File System Access & 18GB Memory Context
- Deep GCP Integration (BigQuery, Cloud Run, AlloyDB, etc.)
- Autonomous Fleet Orchestration (Playwright, SSH Bridges, MCPs)
